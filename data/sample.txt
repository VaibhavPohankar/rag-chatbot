Docker is a containerization platform used to package applications and their dependencies.
Docker images are lightweight, portable files that contain application code and runtime.
Docker containers are running instances of Docker images.

Kubernetes is a container orchestration system used to manage containerized applications.
Pods are the smallest deployable units in Kubernetes.
Deployments in Kubernetes manage application updates and scaling.

AWS is a cloud platform that provides scalable computing services over the internet.
EC2 is a virtual server service provided by AWS.
S3 is an object storage service used to store and retrieve data in AWS.
IAM is a service used for identity and access management in AWS.
CloudWatch is a monitoring service provided by AWS.

Cloud computing is the delivery of computing services over the internet, including servers, storage, and networking.
Computing refers to the use of computers to process data and perform calculations.

Terraform is an infrastructure as code tool used to provision and manage cloud resources.
Infrastructure as code is a practice used to manage and provision infrastructure through code instead of manual processes.
Terraform uses configuration files written in HashiCorp Configuration Language.
Terraform allows automation of infrastructure provisioning.

DevOps is a set of practices that combines software development and IT operations to improve collaboration and automation.
CI/CD is a process that automates software integration, testing, and deployment.

Linux is an open-source operating system widely used in servers.
Cron jobs are used to schedule tasks in Linux systems.

Load balancing is a technique used to distribute traffic across multiple servers.
High availability ensures systems remain operational with minimal downtime.
Auto Scaling automatically adjusts computing resources based on demand.

Monitoring is the process of tracking system performance and health.
Prometheus is a monitoring tool used to collect metrics.
Grafana is a visualization tool used to display monitoring data.

Networking is the practice of connecting systems to enable communication.
DNS is a system that translates domain names into IP addresses.
TCP/IP is a communication protocol used for networking.

A virtual machine is an emulation of a physical computer.
Containers are lightweight alternatives to virtual machines.

An API is an interface that allows communication between software systems.
REST API is a type of API that uses HTTP methods like GET, POST, PUT, and DELETE.

A database is a system used to store and manage data.
SQL is a language used to query relational databases.
NoSQL databases store data in non-tabular formats.

Optimization is the process of improving system performance and efficiency.
Scalability is the ability of a system to handle increasing load.
Latency is the delay between a request and response in a system.

Security is the practice of protecting systems and data from threats.
Encryption is the process of converting data into a secure format.
Authentication verifies the identity of a user or system.
Authorization determines access permissions.

Infrastructure refers to the underlying systems and resources required to run applications.
